# Screenshots

This directory contains screenshots demonstrating OpenQC-VSCode features.

## How to Generate Screenshots

### Prerequisites
1. Open VSCode with OpenQC-VSCode extension installed
2. Open example files from `examples/` directory
3. Use a consistent color theme (recommend: Dark+)

### Screenshot List

#### 1. File Type Detection
- Open various quantum chemistry files

- **Filename**: `feature-detection.png`

#### 2. Syntax Highlighting

- **Filename**: `feature-syntax.png`

#### 3. 3D Structure Visualization
- Water molecule in 3D viewer

#### 4. Data Plotting

- **Filename**: `feature-plotting.png`

#### 5. Input Preview

- **Filename**: `feature-preview.png`

#### 6. LSP Integration
- Hover information
- Error diagnostics
- **Filename**: `feature-lsp.png`

#### 7. Command Palette
- Show all OpenQC commands
- **Filename**: `feature-commands.png`

## Screenshot Guidelines

- Resolution: 1920x1080 or higher
- Format: PNG with transparency disabled
- Theme: Dark+ (for consistency)
- Annotations: Use VSCode built-in screenshot annotations
